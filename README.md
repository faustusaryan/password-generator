# Password Generator

This is a CLI-based strong password generator built using Python.

## Features
- Minimum length validation
- Mandatory:
  - 1 Uppercase letter
  - 1 Digit
  - 1 Symbol
- Random character generation
- Shuffle logic for unpredictability

## Concepts Used
- random module
- string module
- loops
- validation
- shuffle

## How to Run

1. Clone the repository
2. Run the file:

python password_generator.py
